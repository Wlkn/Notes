There are multiple ways to set color:
Hex codes -

Rgb values -
rgb(27, 218, 118)

Color names-
Pretty straightforward

Hsl Values - Stands for Hue, Saturation and Lightness
Hue is the color in the color wheel, Saturation stands
for the amount of grey in the color and lightness is
the amount of black present in the color
